== Description ==

ST EZ Online Library is a simple plugin to help users to easily create a online library of documents. Visitor can browse through these documents and can download them.

== Installation ==

1. Upload folder `st_ez_online_library` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Upload documents / Select document from media library in the EZ Online Library 
1. Upload a preview thumbnail for display.
1. Put shortcode [ez_online_library] on the page where you want to display the library.

== Frequently asked questions ==
